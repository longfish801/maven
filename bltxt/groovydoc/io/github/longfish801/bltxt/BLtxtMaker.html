Package: io.github.longfish801.bltxt

[Groovy] Class BLtxtMaker

    • io.github.longfish801.bltxt.BLtxtMaker
    • All Implemented Interfaces and Traits:
      io.github.longfish801.bltxt.BLtxtMakerIF

      @Slf4j('LOG')
class BLtxtMaker
extends Object

      bltxt記法の文字列の解析にともない、各要素を生成します。
      解析は io.github.longfish801.bltxt.parser.BLtxtParserで実施します。
      BLtxt文書のインスタンスを生成したい場合はBLtxtを参照してください。

      0.3.00 2021/12/29
