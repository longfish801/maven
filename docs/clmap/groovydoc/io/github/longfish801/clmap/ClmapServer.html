      • Method Detail

        • Object cl(String clpath)

          クロージャパスの参照対象を返します。
          パス区切り文字で分割した先頭の名前と、残りのパスに分割します。
          このとき、統語的にありえないパスの場合は例外を投げます。
          残りのパスがなければ、名前が一致する宣言を返します。
          残りのパスがあれば、それを新たなパスとして名前が一致する宣言に依頼します。
          上記で該当するハンドルがなければ nullを返します。

          Parameters:
          clpath - クロージャパス
          Returns:
          クロージャパスの参照対象（存在しない場合はnull）
          Throws:
          TpacHandlingException 統語的にありえないクロージャパスです
