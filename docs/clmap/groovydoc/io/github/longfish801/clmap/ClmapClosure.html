        • Object call(Object[] args)

          クロージャを実行します。
          クロージャを未作成であれば、作成してメンバ変数 closureに格納します。
          Throwableをキャッチしたならば ClmapCallExceptionに置き換えます。
          ただし ClmapCallException自身を置き換えることはしません。

          throws:

          実行結果

        • Closure createClosure()

          クロージャを生成します。
          クロージャのソースコードを未作成であれば、作成してメンバ変数 codeに格納します。
          ソースコードのコンパイルは GroovyShell.evaluateを用います。
          このときクロージャの名前は getClosureName()から取得します。
          クロージャ生成時、Clmap, ClmapMapクラスのメンバ変数 propertiesを 大域変数として利用できるよう設定（delegateに設定）します。
          同じ大域変数名がある場合はより下位の ClmapMapクラスでの値で上書きされます。
          コンパイル時に Throwableをキャッチしたならば WARNログを出力します。

          Returns:
          クロージャ
          See Also:
          createCode()
          getClosureName()

        • String getClosureName()

          クロージャの名前を返します。
          クロージャパスに以下の加工を施した文字列を返します。

          1. 区切り文字 "/"を "_"に、"#"を "-"にすべて置換します。
          2. 接頭辞として "CLMAP"を付与します。
              3. Returns:
              クロージャの名前
