      • Property Detail

        • static Map copyMethods

          コピーモードとコピー処理のクロージャとのマップです。
          クロージャは第一引数にコピー元URLを、第二引数にコピー先ファイルを指定します。
          コピーモードに応じて、以下のとおり動作します。

          overwrite
          常に上書きコピーします。
          modified
          コピー先が存在しないか、コピー元ファイルの更新日時が コピー先ファイルより新しいときだけコピーします。
          notexists
          コピー先が存在しないときのみコピーします。
